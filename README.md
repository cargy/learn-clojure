# Learn Clojure
Some notes from my journey of becoming brave and true while learning Clojure.

## Starting a new project

1. `mkdir new-proj`
2. `cd new-proj`
3. `touch deps.edn`
4. edit `deps.edn` and add the following: 
       
       {:deps {org.clojure/clojure {:mvn/version "1.10.3"}}

## Emacs shortcuts
